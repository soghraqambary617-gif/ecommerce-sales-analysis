# ecommerce-sales-analysis
Data analysis project exploring sales trends, customer behavior, and revenue
Analyze e-commerce data to identify sales trends, top products, and key markets to improve business performance.
Tools Used
Python (pandas, matplotlib)
Jupyter Notebook
Key Insights
Revenue peaks in November and December, indicating strong seasonal demand
The United Kingdom generates the majority of revenue
The top product is Paper Craft, Little Birdie
A small number of products drive most of the revenue
